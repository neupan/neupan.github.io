VSCode开发必备

发表于 2020-02-17

首先下载vscode，可以直接从官网下载，速度很快：下载链接

傻瓜式安装即可

目前已经安装的插件列表

  • Chinese 支持中文显示
  • Code Runner 支持vscode内运行一些代码
  • Beautify 代码美化插件(可以用自带的快捷键command+k+d)
  • vscode-icons 使vscode可以带上各种文件的缩略图，方便查找
  • Markdown All in one
  • Markdown Preview GitHub Styling
  • Debugger for Chrome 可以在vscode里直接调试前端代码(目前还没用上)
  • Path Intellisense 自动补全路径
  • npm 使vscode支持nodejs插件的安装
  • npm Intellisense
  • Project Manager 使vscode打开新文件夹时不用另外新开一个窗口
  • Visual Studio Keymap 使vscode支持vs的快捷键
  • Python 使vscode可以运行Python代码

常用的技巧

  • shift+command+P 打开命令面板
  • Ctrl+` 在vscode内打开(或关闭)终端

    点击1可以创建新的终端，点击2关闭当前终端(可以同时使用多个命令行工具)

常见的问题

  • 目前刚开始遇到vscode中终端乱码，由于我的终端使用了iTerm2安装了其他的字体样式Meslo LG M for Powerline，因此需要修改一下用户配置，打开设置->搜索[terminal]->找到[font family]->设置为对应的Meslo LG M for Powerline即可
